Technical Website Details

This website is built using Flask, a lightweight web framework for Python. It uses Jinja2 templating to render HTML pages and is styled with CSS. The objective of this website is The program should not weigh more than 1MB in absolute size, in order to have a 90s style website and be super fast.

  • Framework: Flask
  • Templating Engine: Jinja2
  • Styling: CSS, no frameworks, no responsive
  • Container: Docker + Docker Compose
  • HTTP Server: Nginx (front-end) + Apache (back-end)
  • Website Size: < 1MB
