Detalles Técnicos del Sitio Web

Este sitio web está construido usando Flask, un framework web ligero para Python. Utiliza Jinja2 para renderizar páginas HTML y está estilizado con CSS. El objetivo de este sitio web es que el programa no pese más de 1MB en tamaño absoluto, para tener un sitio web de estilo de los 90s y ser súper rápido.

  • Framework: Flask
  • Motor de Plantillas: Jinja2
  • Estilos: CSS, sin frameworks, sin responsivo
  • Contenedor: Docker + Docker Compose
  • Servidor HTTP: Nginx (front-end) + Apache (back-end)
  • Tamaño del Sitio Web: < 1MB

Personal

No hay mucho que decir sobre mí, no soy muy extrovertido ni sociable, simplemente hago lo que me gusta, que es el lado de IT y todo lo relacionado con tecnología y telecomunicaciones.

Soy chileno, me gustan los gatos y perros que no sean muy energéticos u obsesivos contigo.

Tengo muchas aficiones, por ejemplo, me gusta el lado de redes, me gusta aprender sobre nuevas tecnologías, me gusta la radio. Las únicas cosas que ya no disfruto mucho son jugar videojuegos y ver televisión. Jugar videojuegos no me genera el mismo "rush" de dopamina que antes; ahora solo la tecnología lo hace.

Y olvídate de la televisión—eso no me genera nada en absoluto.
