# AstronautMarkus / Marcos Reyes Website

A 90s-style personal website for showcasing my projects and skills, built with HTML, CSS, and JavaScript.

No frameworks or libraries were used in the development of this website, as I wanted to create a simple and nostalgic experience for visitors. The design is inspired by the aesthetics of the 90s, with bright colors, pixelated graphics, and a playful layout.
